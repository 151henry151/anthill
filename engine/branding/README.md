# Anthill menu branding textures

These PNGs replace the stock **main menu header** and **About** logo after install (`engine/build.sh` copies them into `$PREFIX/share/luanti/textures/base/pack/`).

They are simple text renders (ImageMagick) so the main menu does not show the upstream wordmark. Replace them with your own artwork if you prefer; match the usual sizes: **`menu_header.png`** 672×72, **`logo.png`** 256×256.
